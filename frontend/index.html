⚙️ Setup

To enable AI features, set your environment variables:

OPENAI_API_KEY=sk-...
DATABASE_URL=sqlite:///cobalt.db
ENVIRONMENT=development

Without a key, the app still works using built-in fallback questions and heuristic scoring.
